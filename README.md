# Django Caching Tutorial

This project demonstrates the use of caching in Django.

## Requirements
- Django >= 3.0
- Python >= 3.6

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/arustcollege-coder/jango-caching-tutorial.git
   cd jango-caching-tutorial
   ```
2. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the server:
   ```bash
   python manage.py runserver
   ```

## Usage
- Access the application at `http://127.0.0.1:8000/`
